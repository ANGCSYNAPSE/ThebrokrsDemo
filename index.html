🏗

Backend stack

  • Frontend: React Native + Kotlin + Swift, Next.js web
  • BFF: Node.js (NestJS), GraphQL federation
  • Gateway: Kong / AWS API Gateway + WAF
  • OLTP: PostgreSQL (RDS), per-service schemas, PITR
  • Doc + Catalog: MongoDB Atlas
  • Cache: Redis (sessions, hot catalog, leaderboard)
  • Search: Elasticsearch (partner, hospital, geo)
  • Analytics: ClickHouse (BI + fraud)
  • Event bus: Kafka (ledger, CDC, fan-out)
  • Infra: EKS Mumbai, Multi-AZ, DR in Hyderabad
  • Payments: Razorpay / PhonePe, T+0 wallet
  • KYC: Setu / Signzy / Digio, Aadhaar tokenized
  • MF: BSE Star / NSE NMF II, SEBI-registered

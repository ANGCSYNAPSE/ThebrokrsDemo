DFD LEVEL 0

System context — everyone and everything connected

TheBrokrs sits between customers, administrators, verified service partners, identity/payment providers, notification channels, and compliance/reporting systems.

DFD LEVEL 1

Customer flow — services, credits, rewards and referrals

The customer never jumps directly from discovery to success. The platform creates a lead, order, or case; checks only required capabilities; tracks asynchronous fulfillment; then calculates rewards.

Important: Money, Service Credit, Reward Points, Voucher/Discount, and Asset Value are separate value types. Rewards become spendable only after the source transaction or referral passes its vesting and fraud window.
DFD LEVEL 2

Partner and admin flow — onboarding to settlement and audit

Partners cannot publish directly. Requests enter role-specific admin queues, sensitive changes require maker-checker approval, and every action produces an audit event.

CONTROLLED STATE MACHINES

States that keep the application understandable

Lead / Order / Case

DRAFT→SUBMITTED→ACTION REQUIRED→PROCESSING→COMPLETED

Alternative terminals: failed, cancelled, refunded, disputed. Every state exposes owner, next action and timeline.

Reward / Credit

CALCULATED→PENDING→VESTED→REDEEMED

Alternative terminals: cancelled, expired, reversed. Source refund or fraud action always links to the compensating ledger entry.

Referral

ATTRIBUTED→SIGNUP→QUALIFICATION→HOLD→VESTED

Alternative terminals: duplicate, ineligible, expired, cancelled, reversed. Launch with single-level referral only.

Partner

APPLIED→VERIFIED→APPROVED→ACTIVE

Operational alternatives: changes required, paused, suspended, terminated. Existing customer records remain accessible.

Payment

CREATED→AUTHORIZED→CAPTURED→RECONCILED

Alternative states: failed, late-authorized, partially refunded, refunded, disputed. Webhook events are idempotent.

Admin case

OPEN→ASSIGNED→IN REVIEW→APPROVAL→RESOLVED

Alternative state: escalated. SLA, reason, evidence, actor, approval and notification remain in immutable history.
